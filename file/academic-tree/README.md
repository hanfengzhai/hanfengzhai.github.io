## Lineage (Math Genealogy Project)

Two paths from Gauss reunite at Runge:

- **Weierstraß line:** Gauss → Gudermann → Weierstraß → Runge
- **Kummer line:** Gauss → Bessel → Scherk → Kummer → Runge

Then: Runge → Born → Oppenheimer → Foldy → Osborn → Yip → Cai → Zhai
